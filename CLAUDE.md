## Project Overview

This is a simple portfolio project. The codebase is in early stages — no framework or build tooling has been set up yet.

## Design Direction

This project uses a frontend design skill (`Front End Design/SKILL.md`) that emphasizes distinctive, production-grade interfaces. Key principles:

- Choose a bold, intentional aesthetic direction — avoid generic "AI slop" (no Inter/Roboto/Arial, no purple-gradient-on-white clichés)
- Typography should be distinctive and characterful; pair a display font with a refined body font
- Commit to a cohesive color palette using CSS variables; dominant colors with sharp accents
- Use meaningful motion and animation — focus on high-impact moments over scattered micro-interactions
- Embrace unexpected layouts: asymmetry, overlap, generous negative space or controlled density
- Match implementation complexity to the aesthetic vision
